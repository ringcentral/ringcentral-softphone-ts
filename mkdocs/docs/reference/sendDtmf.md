# callSession.sendDTMF(char)

Transmits a tone as if a user clicked a button on the dialpad. As you might
expect, acceptable values are:

- `0-9`
- `*`
- `#`

## Inputs

| Parameter   | Description                    |
| ----------- | ------------------------------ |
| `character` | A single character to transmit |

## Outputs

None.
